# Phase 4: 怪物与 Boss 资产

**类型**: animation
**状态**: pending
**依赖**: Phase 3 (英雄作为风格参考)

## 目标

使用 Leonardo.AI 生成 3 种怪物和 1 个 Boss 的部件，并在 DragonBones 中制作动画。保持与英雄统一的美术风格。

## User Stories

### US-014: 使用 Leonardo.AI 生成怪物资产
**输出目录**: `assets/sprites/monsters/`

- [ ] 生成 3 种不同怪物：史莱姆(绿)、蝙蝠(紫)、骷髅(白)
- [ ] 每种怪物的分层部件（身体、眼睛、四肢等）
- [ ] 风格与英雄统一
- [ ] 透明背景 PNG
- [ ] 保存到 `assets/sprites/monsters/`

**怪物设计**:
| 怪物 | 颜色 | 部件数 | 特点 |
|------|------|--------|------|
| 史莱姆 | 绿色 | 3-4 | 简单弹性体，大眼睛 |
| 蝙蝠 | 紫色 | 5-6 | 翅膀、身体、耳朵 |
| 骷髅 | 白色 | 6-8 | 头、身、四肢 |

---

### US-015: 制作怪物骨骼动画
**输出目录**: `assets/sprites/monsters/`

- [ ] 每种怪物包含动画：idle (循环)、walk (循环)、hurt、death
- [ ] 死亡动画结束后保持最后一帧
- [ ] 导出骨骼数据到 `assets/sprites/monsters/`
- [ ] 文件命名：`slime_ske.json`, `bat_ske.json`, `skeleton_ske.json` 等

**动画规格**:
| 动画 | 帧数 | 时长 | 循环 |
|------|------|------|------|
| idle | 4-8 | 0.8s | 是 |
| walk | 6-8 | 0.6s | 是 |
| hurt | 3-4 | 0.2s | 否 |
| death | 8-12 | 0.8s | 否 |

---

### US-016: 使用 Leonardo.AI 生成 Boss 资产
**输出目录**: `assets/sprites/boss/`

- [ ] Boss 设计：大型红色恶魔或龙
- [ ] 部件更多更细致（双角、翅膀、尾巴等）
- [ ] 尺寸是普通怪物的 2-3 倍
- [ ] 风格与其他角色统一
- [ ] 保存到 `assets/sprites/boss/`

**Boss 部件清单**:
- 头部（含双角）
- 身体
- 左翅膀、右翅膀
- 左臂、右臂
- 尾巴
- 腿部（可选）

---

### US-017: 制作 Boss 骨骼动画
**输出目录**: `assets/sprites/boss/`

- [ ] 动画：idle (循环)、attack、hurt、death、rage (低血量状态)
- [ ] 动画比普通怪物更复杂、更有气势
- [ ] 导出到 `assets/sprites/boss/`

**动画规格**:
| 动画 | 帧数 | 时长 | 循环 | 特点 |
|------|------|------|------|------|
| idle | 10-15 | 1.2s | 是 | 威风凛凛 |
| attack | 8-10 | 0.6s | 否 | 挥爪/喷火 |
| hurt | 4-6 | 0.3s | 否 | 轻微后退 |
| death | 15-20 | 1.5s | 否 | 华丽倒下 |
| rage | 12-15 | 1s | 是 | 燃烧效果 |
